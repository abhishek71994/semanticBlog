# RESTful blog :rocket:

This blog is completely restful. It uses

* NodeJS
* Express

For route rendering we use ejs.

## Guide to run :

``` 
$ git clone https://github.com/abhishek71994/semanticBlog
$ cd blogsolutions
$ npm install
$ node app.js
```
